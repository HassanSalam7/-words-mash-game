  🔧 Key Files Explained:

  Core Application:

  - src/app/page.tsx - Main game client with Socket.IO connection logic
  - server/index.js - Socket.IO server handling multiplayer game logic
  - .env.local - Contains NEXT_PUBLIC_SOCKET_URL=http://localhost:4569

  Game Components:

  - Entry/Lobby: EnhancedEntryScreen.tsx, WaitingRoom.tsx
  - Gameplay: ChallengeScreen.tsx, TranslationChallengeScreen.tsx
  - Multiplayer: PrivateRoom.tsx, MultiplayerConversationScreen.tsx
  - Single Player: SinglePlayerStoryScreen.tsx, SinglePlayerTranslationScreen.tsx

  Configuration:

  - next.config.ts - CORS headers and dev origins
  - server/package.json - Socket.IO 4.8.1 dependency
  - package.json - Next.js 15.3.4, Socket.IO client 4.8.1
